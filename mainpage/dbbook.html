책 소개 : 『데이터베이스 배움터』는 데이터베이스의 기본적인 개념과 실용적인 이론을 중심으로 한 학기 동안 가르치기에 적절한 주제들을 정선하여 저술하였고, 개념을 알기 쉽게 보여주는 그림들을 많이 사용하여 이해하기 쉽도록 구성하였다. 대학에서 실습 환경을 마련하기 쉬운 MS SQL Server를 기반으로 책을 저술하였다. 한 장에서 MS SQL Server를 설명하고, 각 장의 필요한 부분에서 MS SQL Server의 중요한 기능을 추가로 포함하였다.
